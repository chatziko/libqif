You will find some LIBQIF use examples on the samples directory.

The library documentation is at the documentation directory. More specifically in the file index.html

The command to compile the example is: 

g++ -I./inc -o samples/sample1 samples/Main.cpp libqif.a

You can find also an script for compile the samples called compile_samples.
